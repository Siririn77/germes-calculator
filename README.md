# Калькулятор сэндвич-панелей «Гермес»

Веб-калькулятор стоимости сэндвич-панелей — одностраничное приложение на чистом HTML/CSS/JavaScript.
Пользователь последовательно выбирает параметры панели, видит расчёт цены в реальном времени,
может сохранить несколько расчётов в историю и скачать результат в Excel.

## Возможности

- **Пошаговый мастер** из 6 этапов с индикатором прогресса:
  1. **Тип панели** — стеновые или кровельные
  2. **Утеплитель** — минеральная вата (МВП) или пенополистирол (ППС)
  3. **Толщина** — доступные значения зависят от типа панели и утеплителя
  4. **Площадь** — ввод в м², минимальный заказ 35 м²
  5. **Цвет** — стандартная палитра + цвета RAL (с наценкой) + декор «SteelArt»
  6. **Профилирование** — 9/5 канавок, гладкий лист, трапеция, микрориф и др. (для стеновых панелей)

- **Расчёт цены в реальном времени** — данные о ценах загружаются с внешнего API (`https://777d.ru/webhook/get-prices-for-script`), цена зависит от площади заказа (до 250 м², до 500 м², свыше 500 м²) и наценки за цвет.

- **История расчётов** — можно сохранить несколько вариантов, раскрыть детали каждого, отредактировать или удалить. Подсчитывается общая стоимость всех расчётов.

- **Экспорт в Excel** — выгрузка всех расчётов в файл `.xlsx` (через библиотеку [SheetJS](https://sheetjs.com/)).

- **Форма заявки** — для скачивания прайс-листа нужно указать номер телефона (с маской ввода `+7 (XXX) XXX-XX-XX`) и согласиться с политикой конфиденциальности.

- **Адаптивный дизайн** — корректно работает на десктопе, планшетах и мобильных устройствах.

- **Анимации интерфейса** — плавные переходы между этапами, анимация появления элементов.

## Использование

### Локальный запуск

Так как это статический HTML-файл, достаточно открыть `Calculator.html` в браузере.
Однако для корректной загрузки изображений и работы API цен рекомендуется запуск через локальный веб-сервер.

**Вариант 1 — через Python:**

```bash
python3 -m http.server 8000
```

Затем открыть в браузере: `http://localhost:8000/Calculator.html`

**Вариант 2 — через Node.js:**

```bash
npx serve .
```

**Вариант 3 — просто открыть файл:**

Дважды кликнуть по `Calculator.html` — откроется в браузере. Функциональность сохранится, но изображение в блоке результатов может не загрузиться при некоторых браузерных ограничениях.

### Размещение на хостинге

Загрузите все файлы проекта на любой статический хостинг (GitHub Pages, Netlify, Vercel, обычный веб-сервер).
Убедитесь, что структура папок сохранена (особенно `Images/`).

## Технические детали

- **Без зависимостей и сборки** — всё в одном файле `Calculator.html`, внешний только CDN: шрифт Google Rubik и библиотека SheetJS (xlsx) для экспорта в Excel.
- **API цен** — при загрузке страницы цены запрашиваются с `https://777d.ru/webhook/get-prices-for-script`. Формат ответа — JSON с разбивкой по типу панели, утеплителю и толщине, с диапазонами цен по объёму (`up250`, `up500`, `over500`).
- **Конфигурация** — минимальная и максимальная площадь настраиваются в константе `CONFIG` внутри файла.
- **Цвета RAL** — палитра цветов задана в массиве `RAL_COLORS` (код, HEX-цвет, название, наценка).

## Настройка

| Параметр | Где находится | Описание |
|---|---|---|
| Минимальная площадь заказа | `CONFIG.MIN_AREA` | По умолчанию 35 м² |
| Максимальная площадь | `CONFIG.MAX_AREA` | По умолчанию 1 000 000 000 |
| Палитра цветов | `RAL_COLORS` | Массив цветов с наценками |
| Доступные толщины | `THICKNESS_LIST` | Зависят от типа панели и утеплителя |
| Варианты профилирования | `PROFILING_OPTIONS` | Разные для стеновых и кровельных |
| URL API цен | `fetchPriceData()` | Адрес запроса прайс-данных |

## Совместимость браузеров

Работает во всех современных браузерах: Chrome, Firefox, Safari, Edge. Для экспорта в Excel требуется интернет (библиотека SheetJS подключается через CDN).
